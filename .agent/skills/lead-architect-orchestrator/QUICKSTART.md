# QuickStart: Architect Mode

1. **Initiate Discussion**: Use `/conversacion` or state "Hablemos de arquitectura...".
2. **Define Goal**: Architect summarizes the vision and trade-offs.
3. **Choose ADR**: If the decision is significant, create an ADR in `docs/architecture/adr/`.
4. **Deploy Tasks**: Update the project `task.md` and delegate to specific skills.

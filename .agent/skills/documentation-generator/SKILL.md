# Documentation Generator Skill

Este skill permite transformar código fuente y metadatos en documentación técnica estructurada, legible y siempre actualizada.

## Áreas de Dominio

### 1. Documentación de Código (In-code)
- **JSDoc/TSDoc:** Estándares para TypeScript y JavaScript.
- **Auto-generación:** Creación de descripciones basadas en tipos de datos y firmas de funciones.

### 2. Generadores de Sitios Estáticos
- **TypeDoc:** Para bibliotecas TypeScript.
- **Docusaurus / VitePress:** Para portales de documentación técnica.
- **MkDocs:** Para documentación basada en Python/Markdown.

### 3. Diagramas como Código (DaC)
- **Mermaid.js / PlantUML:** Integración de diagramas en archivos Markdown.
- **Esquemas de Arquitectura:** Generación de visualizaciones a partir de la estructura de archivos.

## Estándares de Escritura
- **Voz:** Profesional, técnica y directa.
- **Formato:** Uso extensivo de bloques de código, tablas y listas de verificación.
- **Internacionalización:** Soporte para generar docs en múltiples idiomas (i18n).
